## 📱 Responsive Design

### Breakpoints

#### Desktop (> 1200px)
- Full layout dengan sidebar
- Semua animasi aktif
- Cursor trail enabled
- Grid 2-3 columns

#### Tablet (768px - 1200px)
- Adjusted padding
- Grid 2 columns
- Smaller fonts
- Simplified animations

#### Mobile (< 768px)
- Single column layout
- Hamburger menu (if needed)
- Touch-optimized
- Cursor trail disabled
- Reduced particle count

### Mobile Optimizations
```css
- Smaller font sizes
- Reduced padding/margins
- Simplified animations
- Touch-friendly buttons (min 44px)
- Optimized images
- Disabled cursor trail
```

## 💡 Tips & Tricks

### Customization

#### Change Colors
```css
/* Edit in styles.css */
:root {
    --primary-glow: #YOUR_COLOR;
    --accent-border: #YOUR_COLOR;
}
```

#### Adjust Particle Count
```javascript
// Edit in particles.js
this.particleCount = 300; // Change number
this.starCount = 800;     // Change number
```

#### Change Formation Speed
```javascript
// Edit in particles.js
if (this.formationTimer > 300) { // 300 = 5 seconds at 60fps
    // Change 300 to adjust timing
}
```

#### Disable Auto Theme Switch
```javascript
// Comment out in particles.js
// this.startThemeSwitcher();
```
